# Contributor Covenant Code of Conduct

This project uses the Contributor Covenant v2.1 to set expectations for participation.

## Our Pledge
We are committed to providing a welcoming, respectful and safe environment for everyone.

## Expected Behavior
- Be respectful and considerate in all communications.
- Report unacceptable behavior to the maintainers.

## Unacceptable Behavior
- Harassment, discrimination, and abusive language are not tolerated.

## Enforcement
Instances of abusive behavior may be reported by contacting the project maintainers via issues or email. Maintainers will review and take appropriate action.
